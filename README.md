# Alduin

Builds for Windows Standalone and WebGL are automated with Github Actions and the WebGL version is deployed on Netlify.

## Prerequisites

- [Unity 6000.4.8f1](https://unity.com/releases/editor/whats-new/6000.4.8f1)

## Getting started

Clone the project:

```

git clone https://github.com/FelixGCliche/alduin.git

```

Or with SSH:

```

git clone git@github.com:FelixGCliche/alduin.git

```

## CI/CD

This template uses [Github Actions](https://docs.github.com/en/actions) to upload builds for Windows x64 Standalone and WebGL.

If you want more information on the workflow and setup for your own repository, refer to the [CI/CD](docs/ci.md) page.
